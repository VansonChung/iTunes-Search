# iTunes-Search

iTunes-Search is an application to fetch music/movie data from [Itunes API](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html).

## Used Technique

- Kotlin
- MVVM
- ViewPager2
- LiveData
- Coroutine
- ViewBinding
- Sealed Class
- Retrofit
- Coil
- Timber
